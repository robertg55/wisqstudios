Basic Printing

Displaying text to the screen

Description:

When you "print" something in java it will display the text in your console. Please check "Setup Environment" page if you don't know where the console is.

Demonstration:

Calling the method System.out.println("Hello World!") will print the text that is inside the brackets.

The following code should print:

Hello World!

There is also the System.out.print("Hello World!") where there's no "ln" and the end of print which means that a new line will not be created after printing the text. Meaning that if you use the print method twice, the first time you have used it there won't be a new line and the second print will just appear at the end of first one.

Special Characters:

Some characters must have special formatting in order to be printed. For example, everything within your brackets is printed, but what if you want to print the bracket character (")? To print those special characters you must put an up to down slash (\) before them. So if we want to print " we would have to type \". Now that the slash is used to represent a special character, it must itself be a special character and represented with a slash before. Here is a list of special characters and how to represent them:

Character How to display it
" \"
\ \\
' \'
New line (like pressing enter) \\
Tab \t
Backspace (like erasing a character) \b

You might have noticed that the single quote (') also made the list even though you don't actually need to represent it as \' when printing it because your text was initially surrounded by double quotes ("). When we print, we print a sequence of characters called a String. That String is surrounded by double quotes. If you wanted to print a single character (char) you can also display it with single quotes. Therefore, if you wanted to print the single character of single quote, you would have to System.out.println('\'').
