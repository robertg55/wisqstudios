Setup Environment

Setting up for the first time.

Requirments:

  • Java JDK: http://www.oracle.com/technetwork/java/javase/downloads/index.html
    If you are using Windows, make sure you download the correct version. x64 if you are on a 64 bit windows or x86 if on a 32 bit one.
    Click here if you're not sure how to check you windows version you have.
  • Eclipse:https://www.eclipse.org/downloads/
    Once again, make sure to install the correct x86 or x64 version depending on your Windows version.
    If asked for which Eclipse version to download, go for "Eclipse IDE for Java Developers" or "Eclipse IDE for Java EE Developers" (EE stands for Enterprise Edition)

Selecting the Workspace:

When opening Eclipse for the first time, it will ask you where to setup your workspace. The workspace is where all your java projects will be stored and also your Eclipse configuration. Remember where you have set your workspace directory as you might need it in the future if you will want to access the source files for your code.

Setting up a new project:

Creating a new project

To create a new project Go in File -> New -> Java Project

You will need to give the project a name. This is the name that you will find if you go to your workspace folder. Select Finish to create the project.


Creating a class

You must next create a class. The class is where your code will be going. Right click the "src" folder inside your project -> New -> Class.

You will need to give a name to your class. By convention, your class names must always start with a capital letter. Also, as this is the first class of your program, it will be your main class. You must select the option "public static void main(String args)". That will create a "main method" and will be where the program will start to read your code. If you don't have it selected and create a class without a main method, the program won't know where to start reading the code and therefore wont run. If creating multiple classes, only one class must have the "main" selected as the program cannot start reading at two different places at once.


Running your project

Your Eclipse should look similar to the next picture. To test some code, inside the main method type in:

System.out.println("Hello World");

You must then click on the green arrow on top to run the code. You will see the result at the bottom of the screen in the Console where it should say "Hello World".

Reset the view

If you ever accidentally close your Project explorer, your console or any other window and can't find how to open them again like in the following picture, simple go to Window -> Perspective -> Reset Perspective. This should reset all the different windows to how it was when you first installed Eclipse.
